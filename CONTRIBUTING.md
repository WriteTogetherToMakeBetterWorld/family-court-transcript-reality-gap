# Contributing to Family Court Reform

Thanks for helping build this living research! Your input keeps it alive.

## Quick Start
- **Read the [Protocol](PROTOCOL.md)**: Understand how we evolve (or dissolve) this work.
- **No direct edits to `full-report.md`**: Work in section files (e.g., `sections/7.3.md`) â†’ Submit PR.
- **All changes logged**: Add your note to [CHANGELOG.md](CHANGELOG.md).

## Guidelines
| Do | Don't |
|----|-------|
| Cite sources clearly | Add unverified stats |
| Use kid-friendly language where possible | Edit without preview |
| Test links in Markdown | Fork without pinging back |
